# Manifiesto de archivos de recolección manual

Este documento lista los nombres oficiales de archivos esperados para la fase `01_recoleccion_manual`.

La nomenclatura conserva la estructura observada en las carpetas locales:

```text
04_descargas_raw/<editorial>/<editorial>_<area>_depurado_cs_final_2024-03_2025-12.csv
```

En algunos casos se conserva una variante específica por trazabilidad, por ejemplo `ev` para Engineering Village, `wos` para Web of Science o `ieee` para IEEE Xplore.

## Nota de trazabilidad

Si un archivo local conserva sufijos técnicos como `_utf8_bom` o `_afiliacion1`, se puede mantener el archivo real con ese nombre siempre que el reporte y este manifiesto documenten la equivalencia. Para GitHub se recomienda usar los nombres limpios mostrados arriba y documentar la codificación en los reportes.
